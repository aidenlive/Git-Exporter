üéØ Usage Flow

  1. User Opens Static Editor
    Sees enhanced file upload section with GitHub option
  2. Clicks "Select from GitHub"
    Modal opens with authentication options
  3. Authenticates
    Public URL or Personal Access Token
  4. Selects Repository & Branch
    Browse repositories (searchable) and select target branch (main/master first)
  5. Browse & Select Files
    Preview files in tree view, check/uncheck files, set custom entry point
  6. Export Selected Files
    ZIP created with only selected files and attached to input
  7. Submit Form
    Deploy static site with GitHub-exported files
